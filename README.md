#  Biblioteca Digital Lectura Viva

**URL del proyecto:** [https://bibliotecadigital-sow3.onrender.com/](https://bibliotecadigital-sow3.onrender.com/)

## Descripción del Proyecto
**Lectura Viva** es una plataforma de biblioteca digital diseñada para revalorizar y facilitar el acceso a la literatura peruana y latinoamericana. El sistema permite a los usuarios explorar un catálogo curado de obras emblemáticas, ofreciendo funcionalidades interactivas de búsqueda avanzada, alquiler y compra de libros. A través de una experiencia de usuario fluida, el proyecto busca digitalizar la gestión bibliotecaria tradicional, conectando a los lectores modernos con los grandes autores de nuestra región en un entorno seguro, organizado y de alto valor cultural.

## Lenguajes y Tecnologías Utilizados
Para el desarrollo integral de esta aplicación web Full-Stack, se emplearon los siguientes lenguajes y herramientas:

* **HTML5**: Estructuración semántica de las páginas, catálogos de libros y formularios de gestión de usuarios.
* **CSS3**: Diseño visual profesional con paleta de colores institucional, tipografías elegantes y maquetación totalmente adaptable (**Responsive Design**).
* **JavaScript (JS)**: Implementación de la lógica de búsqueda avanzada en el cliente, filtrado dinámico de libros por autor/precio y manejo asíncrono de procesos de alquiler.
* **Python**: Lenguaje principal del backend, utilizando el framework **Flask** para la gestión de rutas, controladores de negocio y renderizado de plantillas.
* **SQL**: Gestión y persistencia de datos mediante el motor relacional **SQLite**, optimizando la velocidad de respuesta en las consultas de inventario y usuarios.
* **JWT (JSON Web Tokens)**: Sistema de seguridad avanzado para el manejo de sesiones protegidas y autenticación robusta de los lectores registrados.
